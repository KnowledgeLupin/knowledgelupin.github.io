# 선형대수학 한국어 교과서 집필 스타일 가이드 v1

## 결정 기록
- 독자 수준: A (입문, 선형대수 첫 과목)
- 교재 용도: B (독학용)
- 설명 톤: A (친절한 대화형)
- 엄밀성 비율: A (직관 70%, 엄밀 30%)
- 증명 정책: B (대부분 정리 증명)
- 이론/계산 비중: B (균형)
- 응용 맥락: D (순수수학 + 공학/물리 + 데이터/ML 혼합)
- 용어 표기: A (한글 우선, 첫 등장만 영문 병기)
- 기호 스타일: A (벡터 굵은 글씨, 예: $\mathbf{v}$)
- 절 구성 템플릿: A (동기 $\to$ 정의 $\to$ 정리 $\to$ 예제 $\to$ 연습)
- 연습문제 정책: B (4단계 난이도 + 힌트)
- 장당 분량: D (유동)

## 1. 집필 목표
이 책은 선형대수 첫 과목을 공부하는 독학자를 대상으로 한다.  
따라서 설명의 1차 목표는 "이해의 진입장벽 최소화", 2차 목표는 "수학적 엄밀성의 습관 형성"으로 둔다.

## 2. 문체와 서술 원칙
1. 독자에게 직접 말을 거는 친절한 대화형 문체를 사용한다.
2. 정의를 제시하기 전에 반드시 동기(왜 필요한가)를 먼저 제시한다.
3. 핵심 아이디어는 직관적으로 설명한 뒤, 짧고 명확한 수식으로 정리한다.
4. 한 문단에는 하나의 메시지만 담는다.
5. 오개념이 자주 생기는 지점은 경고 박스로 명시한다.

## 3. 엄밀성과 증명 기준
1. 직관 70%, 엄밀 30%의 비율을 유지하되, 정리 자체는 정확한 서술을 유지한다.
2. "대부분 정리 증명" 원칙을 적용한다.
3. 단, 계산 반복이 길어지는 경우에는 본문은 핵심 논리만 제시하고 계산은 보조 설명으로 분리한다.
4. 증명은 아래 3단 구조를 따른다.
   - 아이디어 한 줄
   - 본 증명
   - 어디에 쓰이는가(활용 맥락)

## 4. 이론-계산-응용 균형
각 절에서 아래 3요소를 가급적 모두 포함한다.
1. 이론: 정의, 정리, 증명
2. 계산: 손계산 가능한 예제 1~2개
3. 응용: 짧은 응용 박스 1개

응용은 전체 책 기준으로 다음 세 축을 고르게 순환한다.
1. 순수수학 맥락(공간 구조, 정리 간 연결)
2. 공학/물리 맥락(좌표변환, 직교분해, 선형 시스템)
3. 데이터/ML 맥락(최소제곱, PCA, 행렬 분해 직관)

## 5. 용어 및 기호 규칙
1. 한글 용어를 우선 사용하고, 첫 등장 시에만 영문 병기한다.  
   예: 선형사상(linear map), 고유값(eigenvalue)
2. 벡터는 굵은 글씨를 사용한다.  
   예: $\mathbf{v}, \mathbf{w}, \mathbf{x}$
3. 행렬은 대문자 이탤릭체를 사용한다.  
   예: $A, B, P$
4. 집합과 공간은 블랙보드 볼드/캘리그래픽을 일관되게 사용한다.  
   예: $\mathbb{R}, \mathbb{C}, \mathcal{V}, \mathcal{W}$
5. 동일 개념의 표기를 장마다 바꾸지 않는다.

## 6. 절(Section) 템플릿 표준
모든 절은 원칙적으로 아래 순서를 따른다.

1. 동기
- 왜 이 개념이 필요한지 문제 상황으로 시작

2. 정의
- 핵심 정의 1~3개

3. 정리
- 핵심 정리와 짧은 해석

4. 예제
- 기초 예제 1개 + 연결 예제 1개

5. 연습문제
- 4단계 난이도와 힌트 제공

## 7. 연습문제 설계 기준
난이도는 4단계로 통일한다.
1. Level 1 (기초 확인): 정의 적용, 직접 계산
2. Level 2 (표준 응용): 정리 직접 활용
3. Level 3 (연결 추론): 두 개 이상 개념 결합
4. Level 4 (도전 확장): 일반화, 반례 구성, 응용 모델링

각 문항에는 힌트를 제공한다.
- 힌트는 정답이 아니라 "출발점"만 준다.
- 힌트 길이는 1~3문장으로 제한한다.

## 8. 장 분량 운영(유동)
장당 분량은 유동으로 하되, 독학 리듬 유지를 위해 다음 범위를 권장한다.
1. 기초 장(1~4장): 25~35쪽
2. 핵심 장(5~8장): 30~45쪽
3. 구조정리 장(9~11장): 30~50쪽
4. 확장 장(12장, 부록): 20~35쪽

## 9. 독학 친화 편집 규칙
1. 각 장 시작: 학습목표 3~5개, 선수지식 체크리스트
2. 각 절 끝: 핵심 요약 5줄, 자가진단 질문 3개
3. 각 장 끝: 개념 지도(무엇이 무엇에 의존하는가)
4. 긴 증명 뒤: "여기서 기억할 것" 요약 박스 삽입

## 10. 품질 체크리스트(원고 검수용)
1. 동기 없이 정의가 바로 나오지 않았는가?
2. 표기가 앞장과 일치하는가?
3. 정리의 가정/결론이 분명한가?
4. 예제가 계산 절차를 생략하지 않았는가?
5. 연습문제 힌트가 과도하게 정답을 노출하지 않는가?
6. 독학자가 막힐 지점을 사전에 안내했는가?
