# 선형대수학 한국어 교과서 집필 스타일 가이드 v2

## 1. 독자와 목표
- 독자: 수학전공 2학년(미적분학 이수), 독학 사용자
- 목표: 계산 중심 입문을 넘어 선형사상 관점에서 구조정리와 스펙트럴 정리까지 완주
- 범위: 선형대수 내부 완결성 중심, 군론/위상은 본문에서 제외

## 2. 문체 이중 레이어(확정)
1. 설명 문단: 경어체 사용
2. 형식 블록(`정의`, `정리`, `증명`): 엄밀한 수학교과서 평서형 사용
3. 한 블록 내부 문체 혼합 금지

## 3. 블록 타입 표준
- `정의`
- `정리`
- `증명`
- `예제`
- `주의`
- `응용`
- `연습문제(A/B/C)`

## 4. 절(Section) 작성 템플릿
모든 절은 다음 순서를 고정합니다.
1. 도입 설명(경어체)
2. 학습목표 3개
3. 선수개념 체크
4. 핵심 내용(정의/정리/증명 + 예제)
5. 주의(최소 2개)
6. 자가진단퀴즈 3문항(계산/개념/짧은 증명)

## 5. 장(Chapter) 작성 템플릿
1. 장 도입
2. 핵심 절 4~5개
3. 응용 1~2개
4. 장 요약
5. 종합 연습문제

## 6. 증명 정책
1. 핵심 정리는 가능한 한 완전증명으로 제시
2. 각 정리 앞에 `증명 전략` 문단 배치
3. 각 증명 뒤에 `의미 해석` 문단 배치
4. 반복 계산은 본문 요지 + 연습문제로 이관

## 7. 연습문제 정책
1. 절당 최소 A 6, B 4, C 2
2. 장당 최소 45문항
3. 해설 미제공
4. 본문 예제 밀도를 높여 B유형까지 자력 풀이 가능하도록 설계

## 8. 용어/표기 규칙
1. 한글 본문 + 첫 등장 영문 병기
2. 정리 번호: `장.절.번호`
3. 벡터/행렬/공간 표기는 장 전체에서 일관 유지
4. 표기 변경이 필요하면 절 도입에서 명시

## 9. 품질 체크리스트
1. 설명 문단 경어체가 유지되는가?
2. `정의·정리·증명` 블록이 엄밀한가?
3. 각 절에 `주의`가 2개 이상 있는가?
4. 선수개념 없이 갑자기 등장한 개념이 없는가?
5. 연습문제 A/B/C 비율을 만족하는가?
6. 핵심 정리 간 의존성이 앞 절/앞 장에 배치되어 있는가?
