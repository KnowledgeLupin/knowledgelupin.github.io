# 선형대수학 교과서 집필 최종 계획안 (확정본)

## 1. 프로젝트 요약
1. 목표 독자: 수학전공 2학년, 미적분학 이수 후 독학 가능 수준
2. 중심 서사: 행렬 계산 $\to$ 벡터공간/선형사상 $\to$ 분해정리/Jordan $\to$ 스펙트럴
3. 문체: 설명 문단은 경어체, `정의·정리·증명`은 교과서형 엄밀 문체
4. 범위: 군론/위상 제외, 선형대수 내부 완결성 집중
5. 해설: 미제공, 대신 예제 밀도와 독학 장치 강화

## 2. 참고문헌 반영 전략
1. Serge Lang: 구조정리 중심 흐름을 Ch9~Ch15에 반영
2. Hoffman--Kunze: 대수적 엄밀성과 정리 의존성 관리를 Ch7~Ch13에 반영
3. 선형대수와 군: Ch1~Ch8 설명체의 현장감과 친절한 도입 방식을 반영
4. 군론 확장 파트는 본서 범위에서 제외

## 3. 공개 인터페이스/타입
1. 블록 타입: `정의`, `정리`, `증명`, `예제`, `주의`, `응용`, `연습문제(A/B/C)`
2. 절 인터페이스: 도입 설명, 학습목표 3개, 선수개념 체크, 핵심 내용, 주의, 자가진단퀴즈 3문항
3. 장 인터페이스: 장 도입, 핵심 절 4~5개, 응용 1~2개, 장 요약, 종합 연습문제
4. 표기 정책: 한글 본문 + 첫 등장 영문 병기
5. 정리 번호: `장.절.번호`

## 4. 문체/서술 정책
1. 설명 문단 경어체
2. 형식 블록 엄밀 평서체
3. 정리 앞 `증명 전략`, 정리 뒤 `의미 해석` 문단 배치
4. 새 정의 뒤 즉시 예제 배치
5. 각 절 `주의` 최소 2개 배치
6. 응용은 해당 장의 직전 정리만 사용

## 5. 최종 목차
- Part I 연립방정식과 행렬 (Ch1~Ch2)
- Part II 벡터공간과 선형사상 (Ch3~Ch8)
- Part III 다항식과 분해정리 (Ch9~Ch12)
- Part IV 정준형과 스펙트럴 이론 (Ch13~Ch15)
- Part V 쌍선형형식과 에르미트형식 (Ch16)
- 부록 A/B/C

## 6. 장절별 집필 구조
1. Ch1~Ch2: 계산 기반 확립(소거법 정당화, 가역성/랭크)
2. Ch3~Ch6: 벡터공간/선형사상 언어 구축
3. Ch7~Ch8: 쌍대공간/행렬식 고밀도 예제 구성
4. Ch9~Ch12: 분해정리 준비 및 완성
5. Ch13: Jordan 존재/유일성과 계산 절차 분리
6. Ch14~Ch15: 내적 구조와 스펙트럴 정리 완성
7. Ch16: 형식 이론 입문 마무리

## 7. 연습문제/학습장치
1. 절당 최소 A 6, B 4, C 2
2. 장당 최소 45문항
3. A: 기초, B: 개념 연결, C: 증명/종합
4. 해설 미제공, 예제 중심 복구 설계
5. 절 말미 자가진단퀴즈 3문항 고정

## 8. 품질 검증 수용 기준
1. 문체: 설명 경어체 100%, 형식 블록 엄밀 톤 100%
2. 의존성: 정리 선행 개념이 반드시 앞 절/앞 장에 존재
3. 정합성: 용어 병기, 기호, 번호 체계 누락률 0%
4. 독학성: 파일럿 독자가 A/B 문제 목표 해결률 달성
5. 오개념: `주의` 블록이 빈출 오류를 충분히 커버
6. 일정: 2학기 30주 완주 가능한 분량 유지
7. 핵심 정리: rank-nullity, Jordan, spectral theorem 논리 연결 완결

## 9. 집필 실행 일정
1. 1단계(2주): 장별 목표/핵심정리/의존성 그래프 확정
2. 2단계(12주): Ch1~Ch8 초고
3. 3단계(12주): Ch9~Ch16 초고
4. 4단계(4주): 문체 통일, 참조 검수, 난이도 보정
5. 5단계(1학기): 파일럿 운영 후 개정 반영

## 10. 가정 및 기본값
1. 독자는 미적분학 이수자
2. 초반은 일반 field, 내적 이후는 $\mathbb{R}/\mathbb{C}$ 중심
3. 군론/위상은 다루지 않음
4. 교재 성격은 "엄밀한 형식 + 친절한 독학 설명" 병행
